The Regional Languages Of India: Dying A Slow Death?

On this Hindi diwas, warm wishes to all the readers, however it is to be re-informed that the topic is not a sensationalized version of some random conspiracy. Today, check once the uniform resource locators (URLs) of various websites, usernames on Social Media platforms, default language in call centers, main language used for restaurant menus, etc, you’ll find one similarity, everything is only and only in Roman Script - you’ll find latin alphabets everywhere and you can’t use letters from Devanagari script in these cases. This is a clear indication of language dictatorship.
